Règles du jeu "Wordle"

Le mot mystère en 6 essais.
Chaque tentative doit porter sur un mot valide.
Appuyer sur la touche Entrée pour deviner.
Après chaque tentative, la couleur des cases change pour indiquer à quel point vous êtes proche de deviner le mot.
Le vert signifie que la lettre est dans la bonne position dans le mot.
Jaune signifie que la lettre est dans le mot, mais pas dans le bonne position dans le mot.
Gris signifie que la lettre a déjà été utilisée.
